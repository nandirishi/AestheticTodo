# 🌸 Aesthetic To-Do List App

A beautifully designed, minimalist To-Do List application built with a custom pastel pink theme. This app focuses on providing a clean user interface and smooth experience for tracking daily tasks.

## ✨ Features
* 📝 **Task Management:** Easily add, track, and delete daily tasks.
* 🎨 **Aesthetic UI:** Customized pastel pink color palette for a visually soothing workspace.
* ⚡ **State Management:** Clean architectural state tracking for seamless data updates.

## 🛠️ Tech Stack
* **Language:** Kotlin / Dart
* **Framework:** Android Studio / Jetpack Compose / Flutter
* **State Management:** Riverpod / State Architecture
